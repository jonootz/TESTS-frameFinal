Test Cases - Filenames

Boundary test all filenames (Numbers, Text, Various characters and symbols including international). Test creating file types with various characters and confirm they appear correctly in the Target Editor and in the output:

  • Topics (Startup topic)
  • Skin
  • TOC (Master TOC)
  • Browse Sequence (be sure to enable it in the skin)
  • Output file
  • Output folder

 

In the Output File field, enter filenames which include file extensions of other output types (ex. For WebHelp output, enter “Output.chm”). Confirm that Flare generates the correct output and the output file has the correct file extension.
